Introduction to CSS

CSS (Cascading Style Sheets) is a powerful styling language used to control the presentation and layout of HTML and XML documents. It separates the content from the design, enabling web developers to create visually appealing and consistent web pages. CSS uses a selector and declaration block syntax: 

    
selector {
    property: value;
}

The selector targets HTML elements, while the property defines the style to be applied, and the value specifies the property's characteristics.

CSS features include:

  • Changing font, color, and size of text.
  • Setting backgrounds and images.
  • Creating layouts and positioning elements.
  • Adding animations and transitions.

You can include CSS directly in the <style> element within the <head> of an HTML document. By using selectors and properties, you can style specific elements or apply styles globally to entire sections.

Types of CSS Selectors

There are several types of selectors in CSS that can be used to target different parts or components.

  • The Universal Selector : selects all elements on the Page. 
    Example :  * {
                property : value;
            }
  • The Element Selector : (h1 or anyTagName) applies specific styles to the <h1> element. 
    Example :  h1 {
                property : value;
            }
  • The Class Selector (.classname) selects elements with class "highlight" to style them. 
    Example :  .classname {
                property : value;
            }
  • The ID Selector (#ID) selects the element with selected ID to apply specific styles. 
    Example :  .ID {
                property : value;
            }
  • The Descendant Selector (div p) selects elements that are descendants of selected Element. 
    Example :  We have a HTML structure of   
            <div class="parent">
                <div class="child">
                    <p>text..</p>    
                </div>
            </div>
             Selector to select "p" tag will be 
.parent p{
            property : value;
        }
  • The Child Selector (ul>li) selects <li> elements that are direct children of <ul> to style their list bullet. 
    Example :  We have a HTML structure of   
            <ul>
                <li>
                    <p>text..</p>    
                </li>
            </ul>
 Selector to select "p" tag will be 
ul > li{
        property : value;
    }

Style rule cascading and inheritance

CSS inheritance refers to the relationship between HTML tags (think parent and children tags) and how certain CSS styles can apply to a tag even though there aren't any CSS rules directly applied to it. Cascading refers to the fact that cumulative styles across multiple CSS rules are applied to each and every HTML tag.

Learn Bootstrap

Bootstrap is a powerful front-end framework for building responsive, mobile-first web applications. It was developed by Twitter and is now maintained by the Bootstrap team at GitHub.


Key Features of Bootstrap

  • Responsive Design: Bootstrap's grid system and components adjust automatically to fit different screen sizes, ensuring your website looks great on various devices.
  • Pre-styled Components: Bootstrap provides a collection of ready-to-use UI components, such as buttons, forms, navigation bars, and more, saving development time and effort.
  • Mobile-first Approach: Bootstrap's CSS is designed to prioritize mobile devices, ensuring a smooth experience for mobile users.
  • Customizable: You can customize Bootstrap's variables and styles to match your brand and design preferences.
  • Extensive Documentation: Bootstrap offers comprehensive documentation and examples to help developers understand and implement its features.
  • Browser Compatibility: Bootstrap supports all modern browsers, ensuring consistent behavior across different platforms.

Getting Started with Bootstrap

To use Bootstrap, include its CSS and JavaScript files in your HTML document:


<!-- Add Bootstrap CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">

<!-- Add Bootstrap JS (optional) -->
<script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.1/dist/umd/popper.min.js"></script>
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

Using Bootstrap Components

Bootstrap provides a wide range of components that you can easily use in your HTML:

<!-- Example of a Bootstrap Button -->
<button class="btn btn-primary">Click Me</button>

Creating a Responsive Layout

With Bootstrap's grid system, you can create responsive layouts that adapt to different screen sizes:

<!-- Example of a Responsive Grid Layout -->
<div class="container">
    <div class="row">
        <div class="col-md-6">Content on the left</div>
        <div class="col-md-6">Content on the right</div>
    </div>
</div>
